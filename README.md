## 🔧 Manual Install (Chrome / Edge / Brave)
1. Clone the repository:
   ```bash
   git clone https://github.com/code-hemu/klak-newtab.git
   cd klak-newtab

2. Install dependencies and build:
   ```bash
   npm install
   npm run build
   
The output will be generated in a build/ folder.
